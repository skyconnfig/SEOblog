操作指南

📝 发布文章

  1. 1
    创建 .md 文件 — 在 src/content/blog/ 下新建文件，文件名即 URL slug（如 my-article.md）
  2. 2
    填写 Frontmatter — 文件顶部用 --- 包裹元数据
  3. ✓
    运行 node scripts/publish.js 选择 3 部署，或手动 git push
--- title: "你的文章标题" description: "SEO 描述" date: 2026-05-20 tags: ["PLG", "SaaS"] draft: false --- 文章正文 Markdown...

📦 上传资源

  1. 1
    把 ZIP/PDF 文件放入 public/downloads/
  2. 2
    把 封面图 放入 public/assets/resource-previews/<slug>/
  3. 3
    编辑 src/data/resources.ts，添加资源条目
  4. ✓
    部署到 GitHub

🚀 部署

  1. 1
    运行 publish.cmd（双击）或 node scripts/publish.js
  2. 2
    选择选项 3: Build & deploy to GitHub
  3. 3
    输入 commit message，自动构建并推送
